# 参考广告搜索模板

目标：为一个商品收集 5-10 条同类信息流广告参考。

## 搜索顺序

1. 海外商品优先：TikTok Creative Center / Top Ads
2. 国内商品优先：巨量创意
3. 竞品品牌补充：Meta Ads Library
4. 官网域名补充：Google Ads Transparency Center
5. 搜不到时：让用户提供竞品名、本地视频或链接

## 实测参考路径

- TikTok：打开 Creative Center，进入 Top Ads；未登录时通常只能看少量广告，但足够验证方向。
- Meta：优先用竞品品牌或页面搜索。比如便携榨汁杯可以搜 `BlendJet`，会看到同类广告。
- Google：优先用官网域名或品牌名搜索。关键词泛搜不稳定，域名更可靠。
- 巨量：国内商品优先搜索中文类目词和痛点词；如果需要登录，保存搜索词和截图，继续用其他平台补充。

## 搜索词生成

- 类目词：商品是什么，例如 `portable blender`、`护肤精华`
- 痛点词：解决什么问题，例如 `dark spots`、`肩颈酸痛`
- 场景词：在哪里用，例如 `travel`、`office`
- 人群词：谁会买，例如 `new moms`、`上班族`
- 竞品词：同类品牌、商品名、官网域名

## 筛选标准

优先选择：

- 前 3 秒钩子明确
- 产品露出清楚
- 卖点单一但强
- 有用户场景或真实使用感
- 有明确行动引导
- 有多个同主题变体或持续投放痕迹

不要选择：

- 纯品牌大片
- 看不清产品的氛围视频
- 跟商品类目无关的视频
- 需要绕过权限才能访问的内容

## 记录格式

```json
{
  "platform": "",
  "url": "",
  "local_file": "",
  "screenshot": "",
  "product_category": "",
  "duration": "",
  "hook": "",
  "main_selling_point": "",
  "cta": "",
  "why_reference": ""
}
```
