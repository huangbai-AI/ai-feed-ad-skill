# 真实视频依据参考脚本库

正式小类必须先在这里有一条参考脚本。没有来源编号、没有反推脚本的小类，不能进入 `commerce_ad_subtypes.md` 和批量候选。

来源只用于内部拆解。最终交给即梦 / Seedance 的提示词不要出现来源路径、平台名、参考视频编号。

## 来源等级

- A：本机真实视频已抽帧，且口播可用。
- B：本机真实视频已抽帧，口播不可用或无口播，按画面反推。
- C：飞书文档内有生成案例、故事板或视频产出截图；视频附件未能下载时，必须写明只按文档可见信息反推。
