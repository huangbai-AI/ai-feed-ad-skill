# AI 带货广告生产类型库

三类是一级分类，不是最终模板。每次必须继续选择一个有来源编号的小类，并读取 `templates/reference_script_library.md` 和 `templates/commerce_ad_subtypes.md`。

## 使用规则

- 默认不搜索、不抓取、不下载外部参考。
- 正式小类必须有真实视频依据和参考脚本。
- 没有参考脚本的小类不能进入默认生成和批量候选。
- 商品图是外观硬约束：颜色、形状、材质、配件、包装不能随意改。
- 一条视频只讲 1 个主卖点。
- 前 1 秒必须出现商品、使用结果、强剧情冲突或产品主视觉。
- 最终提示词不能出现“参考某视频”“照着某平台”“来源编号”等字样。
- 借鉴影视感时，只借结构、情绪节奏和镜头语言，不复制真实剧名、角色名、台词、音乐、服化道。

## 大类 1：剧情广告

适合：

- 商品需要情绪反转、短剧情、状态变化或强钩子。
- 用户要求“剧情广告、短剧带货、反转、故事感”。

当前正式小类：

| 小类 | 来源编号 | 适合商品 | 核心结构 |
| --- | --- | --- | --- |
| 情感反转剧情 | LD-01 | 墨镜、香水、包袋、服饰、首饰、礼品 | 情绪低点 -> 商品触发变化 -> 新状态定格 |

待反推但不进默认库：

- 武侠闯江湖、宫廷斗心机、重生开大招、反转停不下、霸总、宫斗。
- 这些词来自飞书文档，但当前没有完成视频下载/观看/脚本反推。用户要求这类风格时，先说明需要对应视频或文档案例，再补参考脚本后使用。

## 大类 2：品牌大片

适合：

- 商品需要高级感、质感、品牌调性。
- 商品本身有视觉价值，例如食品饮料、香水、包袋、鞋服、首饰、数码、美妆。

当前正式小类：

| 小类 | 来源编号 | 适合商品 | 核心结构 |
| --- | --- | --- | --- |
| 产品英雄广告 | LV-05 | 食品饮料、香水、美妆、包装强单品 | 产品极近特写 -> 感官卖点 -> 正面定格 |
| 电影感 TV 广告 | LV-06 | 巧克力、零食、酒水、香水、礼品 | 强氛围世界观 -> 角色体验 -> 产品定格 |
| 时尚生活方式跟拍 | LV-08 | 包袋、服装、鞋、眼镜、饰品 | 真人跟拍 -> 商品细节 -> 生活方式定格 |

待反推但不进默认库：

- 冷暖科技风、动感潮流风、高级简约风。
- 这些词来自飞书文档截图，当前作为视觉风格词，不作为独立小类。

## 大类 3：达人带货

适合：

- 大多数电商转化视频。
- 需要真实感、口播感、试用感、上身效果或使用证明。

当前正式小类：

| 小类 | 来源编号 | 适合商品 | 核心结构 |
| --- | --- | --- | --- |
| UGC 手持口播展示 | LV-01 | 手机壳、美妆、小配件、杯子、桌面好物 | 真人自拍 -> 指出细节 -> 推荐理由 |
| 开箱揭晓 | LV-02 | 礼品、玩具、盲盒、包袋、鞋、首饰 | 包装开场 -> 取出商品 -> 同框定格 |
| 使用场景证明 | LV-03 | 小电器、厨房用品、清洁用品、工具 | 痛点场景 -> 使用动作 -> 结果证明 |
| 试穿前后变化 | LV-04 | 服装、鞋、包、眼镜、首饰、美妆 | 未上身 -> 上身变化 -> 细节判断 |
| 俯拍 ASMR 开箱 | LV-07 | 首饰、礼盒、香水、精品小物 | 俯拍开盒 -> 取出细节 -> 仪式感定格 |

待反推但不进默认库：

- 家居好物、穿搭单品、美妆护肤、休闲零食、数码配件、母婴用品、宠物用品。
- 这些是飞书文档提到的赛道，不是已经反推完成的小类。

## 爆款表达结构

这些是钩子写法，不是生产类型。

| 结构 | 公式 | 适合 |
| --- | --- | --- |
| 信息差 | 现状否定 + 工具/商品揭秘 + 方法讲解 | 教程、工具、低门槛生成 |
| 机遇焦虑 | 别人已跑通 + 你还在观望 + 动作引导 | AI 带货、商家、大促 |
| 视觉震撼 | 先给效果 + 成本揭秘 + 安利 | 品牌大片、视觉强商品 |
| 亲身经验 | 结果展示 + 核心经验 + 可复制步骤 | 达人、店主、教程 |
| 保姆级教程 | 分步演示 + 结果交付 | 小白教程、操作类 |
| 痛点共鸣 | 翻车现场 + 原因 + 解决方案 | 功能型商品、避坑型商品 |

## 5-10 秒短变体规则

```text
0-1 秒：商品、结果、冲突或主视觉必须出现。
1-4 秒：完成一个清楚动作，不做多场景混剪。
4-8 秒：给卖点证明或细节近景。
8-10 秒：商品定格或轻 CTA。
```

短变体不要写复杂剧情，不要同时讲多个卖点。
