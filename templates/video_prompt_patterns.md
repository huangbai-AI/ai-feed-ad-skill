# 商品广告视频提示词模式库

本文件沉淀自 Higgsfield Product Ad Generator 页面 24 条公开参考视频的画面结构拆解。使用时只学习结构、节奏和镜头语言，不复制人物、品牌、包装、台词或具体画面。

## 使用方法

生成镜头提示词前，先按商品和卖点选择 1 个主模式；一条 15-30 秒广告最多混合 2 个模式，避免视频风格发散。

如果用户要求 5 秒视频、5 秒变体、一次生成多条短视频，使用“5 秒单场景变体规则”，不要套 15-30 秒完整广告结构。5 秒视频必须具体到时间段，不写泛泛的“展示卖点”“高级感”“种草感”。

参考来源只用于内部拆解。最终给视频生成器的提示词不能出现“参考 xx”“来源视频”“Higgsfield”“照着这个视频”等字样。

用户指定人物约束时，必须作为全局硬约束继承到每条提示词，例如：亚洲女生、亚洲男生、欧美女生、25-35 岁、通勤穿搭、运动穿搭。

输出每个镜头时必须写明：

- 主模式：如 `ugc_handheld_demo`
- 画面目标：钩子 / 展示 / 证明 / 转化
- 商品出现方式：手持 / 开箱 / 使用中 / 上身 / 宏观特写
- 镜头运动：推进 / 环绕 / 跟拍 / 手持轻晃 / 俯拍静物
- 真实场景：卧室、浴室、厨房、健身房、球场、街头、影棚

## 选型表

| 商品类型 | 优先模式 | 原因 |
| --- | --- | --- |
| 手机壳、小电器、美妆、杯子 | `ugc_handheld_demo` | 信息流感强，商品和真人可信度高 |
| 玩具、礼品、包袋、鞋服、首饰 | `unboxing_reveal` | 开箱天然有悬念，适合做前 3 秒钩子 |
| 榨汁杯、电动牙刷、健身器材、厨房用品 | `use_case_proof` | 能把功能直接拍成可见结果 |
| 衣服、鞋、包、眼镜、配饰 | `try_on_transformation` | 前后变化明显，适合快节奏种草 |
| 饮料、零食、香水、包装好看的产品 | `macro_product_hero` | 适合做高质感产品主视觉 |
| 高客单礼品、巧克力、手表、香水 | `cinematic_tv_spot` | 提升价值感和送礼感 |
| 手工、贴纸、收纳、DIY 工具 | `tutorial_step_demo` | 步骤明确，适合解释“怎么用” |

## 类目到参考模式组合

当用户提供本地参考视频时，先按类目选最接近的画面结构，再改成用户商品。不要把不相关类目的画面硬套到商品上。

| 商品类目 | 优先参考结构 | 避免 |
| --- | --- | --- |
| 女包、斜挎包、手提包 | 桌面开箱、床上礼盒开包、俯拍开箱、自拍手持展示、卧室试背、完整穿搭、街头跟拍、包身细节 | 食品爆炸特效、电器操作过程、运动器械组装 |
| 鞋服、配饰 | 手持单品、跳切上身、全身展示、材质近景、街拍跟拍 | 纯静物摆拍过长 |
| 美妆、杯子、小配件 | 真人手持近镜头、手指指出细节、打开/涂抹/使用、真人反馈 | 电影感暗光过重 |
| 食品饮料 | 宏观特写、水珠/配料/口感元素、产品定格 | 复杂真人剧情 |
| 小电器、厨房用品 | 真实使用过程、按钮/安装/操作特写、结果画面 | 只拍包装不展示功能 |

女包类 8 条 5 秒变体优先顺序：

1. 桌面开箱。
2. 床上礼盒开包。
3. 俯拍开箱。
4. 自拍手持展示。
5. 卧室变装试背。
6. 穿搭完成版。
7. 街头时尚跟拍。
8. 卧室包袋上身展示。

## 模式 1：UGC 手持口播展示

适合：手机壳、美妆、牙刷、杯子、小电器、运动用品。

反推结构：

1. 自拍近景开场，人物直接拿商品靠近镜头。
2. 1-2 个商品细节特写，手指指向关键卖点。
3. 切到真实使用场景，展示使用动作。
4. 回到人物半身，给出结果感或推荐语。

提示词模板：

```text
竖屏 9:16，真实手机自拍风格。一位自然上镜的年轻用户站在{真实场景}，手持{商品}靠近镜头，前 1 秒直接展示商品正面。镜头轻微手持晃动，光线自然，画面像 TikTok/抖音真实种草视频。随后给{商品细节}近景，人物用手指指出{核心卖点}，再切到{使用动作}，最后人物看向镜头露出满意表情。节奏快，真实，不要广告大片感。
```

负面要求：

```text
不要夸张表演，不要虚假品牌标志，不要多余道具，不要让商品变形，不要出现不可读文字。
```

## 模式 2：开箱揭晓

适合：玩具、礼品、包袋、鞋、首饰、盲盒、大件产品。

反推结构：

1. 包装盒占满画面，手进入画面准备打开。
2. 拆封、掀盖、取出商品，保留手部动作。
3. 商品完整亮相，停留 1 秒让观众看清。
4. 商品和包装同框，强化“收到货”的真实感。

提示词模板：

```text
竖屏 9:16，真实开箱视频。镜头从{包装盒/快递箱/礼盒}的近景开始，双手进入画面慢慢打开包装，动作清晰有期待感。打开后露出{商品}，镜头轻微推进到商品细节，手把商品拿出来转向镜头展示{卖点细节}。背景是{卧室/桌面/床铺/健身房}，自然光，像用户刚收到货后马上拍的开箱视频。结尾让商品和包装同框摆放。
```

负面要求：

```text
不要把开箱拍成纯静物，不要让手指遮住商品，不要让包装文字乱生成，不要改变商品颜色和形状。
```

## 模式 3：使用场景证明

适合：厨房小电器、电动牙刷、健身器材、运动用品、清洁工具。

反推结构：

1. 人物拿起商品说明“要解决什么”。
2. 马上进入真实使用动作。
3. 给出过程特写：按钮、搅拌、刷头、运动、出汁、安装。
4. 结果画面：成品、上脚、骑行、清洁后、使用后表情。

提示词模板：

```text
竖屏 9:16，真实生活方式广告。人物在{厨房/浴室/健身房/运动场}中拿起{商品}，镜头先给商品近景，再快速切到实际使用：{关键使用动作}。画面包含 2 个过程特写：{细节一}、{细节二}，最后展示清楚的结果：{使用结果}。整体像真实用户记录，节奏紧凑，商品始终清楚可见。
```

负面要求：

```text
不要只拍人物不拍商品，不要跳过使用过程，不要让结果和卖点无关，不要出现危险或不合理使用方式。
```

## 模式 4：试穿/前后变化

适合：服装、鞋、包、眼镜、配饰、美妆。

反推结构：

1. 人物拿着未穿戴的商品出镜。
2. 手遮镜头、转身、跳切或自然动作完成变化。
3. 展示上身/上脚/上脸效果。
4. 镜头推近到材质、版型、颜色或搭配细节。

提示词模板：

```text
竖屏 9:16，真实试穿变装视频。人物在{卧室/更衣区/街头/浴室镜前}手持{商品}面对镜头，先展示未使用状态。通过一次自然转身或手遮镜头完成前后变化，下一秒人物已经穿戴好{商品}。镜头从全身效果切到{材质/颜色/版型/配饰细节}近景，再回到人物自信站姿。风格真实、干净、节奏快，适合信息流。
```

负面要求：

```text
不要夸张变身特效，不要改变人物身份，不要让服装穿帮，不要让商品图案变形。
```

## 模式 5：宏观产品英雄镜头

适合：饮料、零食、香水、鞋、包装强的产品。

反推结构：

1. 商品在纯色或强视觉背景中占据中心。
2. 快速宏观特写：水珠、气泡、碎片、材质、配料、鞋面。
3. 商品悬浮、旋转、穿越或慢动作落位。
4. 结尾是干净的产品定格画面。

提示词模板：

```text
竖屏 9:16，高质感产品英雄镜头。{商品}位于画面中心，背景是{颜色/场景氛围}，镜头从极近特写开始，突出{材质/水珠/纹理/配料/包装细节}。随后商品缓慢旋转或轻微悬浮，周围出现与卖点相关的元素：{元素一}、{元素二}。镜头运动顺滑，有广告质感，最后定格在商品正面，画面干净，让包装和外形清楚可见。
```

负面要求：

```text
不要复杂剧情，不要人物抢戏，不要过多文字，不要让商品商标和包装变形。
```

## 模式 6：电影感 TV 广告

适合：礼品、巧克力、手表、香水、包袋、高客单产品。

反推结构：

1. 场景先建立氛围：暗光、宝箱、卧室、礼盒、街头。
2. 人物以仪式感方式拿出商品。
3. 商品特写配合慢动作和光影。
4. 结尾给一句简短品牌感字幕或购买理由。

提示词模板：

```text
竖屏 9:16，电影感商品广告。场景是{氛围场景}，灯光有层次，背景简洁。人物以有仪式感的动作打开{礼盒/盒子/包裹}，慢慢取出{商品}。镜头切到商品近景，突出{高级材质/精致细节/送礼感}，再切回人物使用或欣赏商品的反应。整体像 15 秒 TV spot，质感高级，但商品必须始终清晰。
```

负面要求：

```text
不要剧情太复杂，不要复古到看不清商品，不要出现真实品牌仿冒，不要生成不可读的大段字幕。
```

## 模式 7：俯拍 ASMR 开箱

适合：首饰、礼品、手工品、小配件、收纳用品。

反推结构：

1. 俯拍桌面或床面，商品包装居中。
2. 双手慢慢打开包装，动作有节奏。
3. 取出商品并摆齐所有配件。
4. 手部近景展示细节和质感。

提示词模板：

```text
竖屏 9:16，俯拍 ASMR 开箱。镜头固定在{桌面/床铺/丝绸布面}上方，{商品包装}居中。双手从画面下方进入，缓慢打开包装，依次取出{商品}和{配件}，每个动作都清楚、干净、有秩序。镜头给商品细节近景，突出{材质/光泽/小巧细节}，最后把商品和配件整齐摆放成完整展示。
```

负面要求：

```text
不要手部动作混乱，不要多手指畸形，不要遮挡商品，不要桌面杂乱。
```

## 模式 8：时尚生活方式跟拍

适合：包袋、鞋、服装、配饰。

反推结构：

1. 人物走入真实场景，商品自然出现在穿搭里。
2. 跟拍步伐、转身、拿包、抬手等动作。
3. 切商品近景：链条、皮革、鞋面、手表、配饰。
4. 回到完整穿搭，建立“用了就更好看”的欲望。

提示词模板：

```text
竖屏 9:16，时尚生活方式广告。人物穿着符合{目标人群}的日常造型，在{街头/卧室/更衣区/咖啡店/健身房}自然走动。{商品}作为穿搭重点清楚出现，镜头先跟拍人物动作，再切到{商品细节}近景，最后回到完整穿搭效果。画面有真实生活质感，轻微手持或稳定跟拍，像社交平台时尚种草视频。
```

负面要求：

```text
不要只拍模特不拍商品，不要让商品变成背景装饰，不要过度摆拍，不要复制参考人物造型。
```

## 组合公式

### 5 秒单场景变体

用于用户要求“5 秒提示词”“一次跑 8 条”“每条一个类型”。

```text
0-0.5 秒：商品、包装、上身效果或明确结果必须已经出现。
0.5-1 秒：建立真实场景和人物动作。
1-3 秒：完成核心动作，例如打开、取出、斜挎、走动、整理、使用。
3-4.5 秒：给商品近景或上身效果，展示 1 个最重要细节。
4.5-5 秒：定格在商品清楚可见的画面，方便后续剪辑或转场。
```

5 秒提示词必须包含：

- 时长和比例：5 秒竖屏 9:16。
- 人物：按用户要求写清人种、性别、年龄感和风格。
- 商品：颜色、材质、配件、形状必须和商品图一致。
- 场景：只选一个具体场景。
- 动作链：最多 2-3 个连续动作。
- 镜头：手持、俯拍、推进、跟拍、近景、定格。
- 负面要求：不要品牌字、不要字幕、不要水印，不要改变商品外观。

不要在 5 秒提示词里写：

- 参考某视频。
- 学习某平台。
- 多个无关场景快速混剪。
- 复杂剧情。
- 超过一个主卖点。

### 15 秒快节奏

```text
0-3 秒：UGC 手持商品或开箱第一眼，直接给钩子。
3-8 秒：展示 1 个最强卖点，必须有手部动作或真实使用。
8-12 秒：给结果或细节特写。
12-15 秒：人物反应或商品定格，加短 CTA。
```

### 30 秒完整种草

```text
0-3 秒：痛点/欲望钩子。
3-8 秒：开箱或产品出现。
8-18 秒：使用过程或试穿变化。
18-25 秒：结果证明和细节补充。
25-30 秒：优惠/购买理由/行动引导。
```

## 生成前检查

- 第一镜头必须在 1 秒内出现商品或明确结果；5 秒视频必须在前 0.5 秒出现。
- 每条视频只讲 1 个主卖点。
- 至少 2 个镜头出现手部动作或真实使用动作。
- 至少 1 个镜头是商品近景。
- 如果是服装、包、鞋、配饰，必须有上身/上脚/佩戴效果。
- 如果是功能型商品，必须有使用过程和结果。
- 最终提示词不能暴露参考来源；参考编号只允许写在内部分析或备注字段。
