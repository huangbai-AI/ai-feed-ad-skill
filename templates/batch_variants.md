# 批量变体模板

样片通过后，再生成批量变体。

默认变体：

- 3 个开场钩子
- 3 个主卖点角度
- 2 种人物表达
- 2 种结尾行动引导

输出格式：

```json
{
  "base_script": "",
  "variants": [
    {
      "variant_id": "v001",
      "angle": "",
      "hook": "",
      "main_change": "",
      "shots_to_regenerate": [],
      "expected_use": ""
    }
  ]
}
```
