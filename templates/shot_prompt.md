# 即梦 / Seedance 提示词模板

把选中的脚本转成可提交给 `dreamina multimodal2video` 或 `image2video` 的提示词。

## 总规则

- 默认 10-15 秒竖屏 `9:16`。
- 默认模型 `seedance2.0fast`。
- 商品图存在时优先 `multimodal2video`。
- 每条提示词必须锁定商品外观。
- 不写参考来源，不写平台名，不写“照着某视频”。
- 内部脚本必须带 `source_id`，但最终 prompt 不写来源编号。
- 不要求模型生成复杂可读文字；字幕建议后期加。
- 生成提交后如果排队，只记录任务 ID，不重复提交。

## 输出格式

```json
{
  "video_ratio": "9:16",
  "duration": 10,
  "production_type": "剧情广告 / 品牌大片 / 达人带货",
  "production_subtype": "",

  "hook_structure": "信息差 / 机遇焦虑 / 视觉震撼 / 亲身经验 / 保姆级教程 / 痛点共鸣",

  "product_lock": {
    "name": "",
    "color": "",
    "material": "",
    "fixed_details": []
  },
  "prompt_file": "shots/shot_001.txt",
  "prompt": "",
  "negative_prompt": ""
}
```

## 通用 Prompt 骨架

```text
生成一条{时长}秒竖屏 9:16 的{生产类型}/{生产子类型}。商品是{商品名}，以输入商品图为唯一外观依据，必须保持{颜色、形状、材质、包装、配件、logo位置等固定细节}。

0-3 秒：{钩子画面}，商品/结果/冲突/主视觉必须已经出现。
3-8 秒：{核心动作或剧情推进}，只证明一个主卖点：{核心卖点}。
8-12 秒：{细节近景、使用结果、上身效果或反转结果}。
12-15 秒：{商品定格或轻 CTA}，商品清楚可见。

画面风格：{真实信息流 / 高级品牌片 / 短剧情广告}。
负面要求：不要品牌乱码，不要水印，不要字幕，不要改变商品外观，不要多余人物，不要复杂无关剧情。
```

## 剧情广告 Prompt 模板

```text
生成一条 15 秒竖屏剧情带货广告。商品是{商品名}，以商品图为唯一外观依据，必须保持{商品固定细节}。0-3 秒，人物在{场景}遇到{痛点/尴尬/误会}，商品或商品相关问题已经出现。3-8 秒，{商品名}自然进入剧情，人物通过{动作}发现{核心卖点}。8-12 秒，剧情出现反转：{反转结果}。12-15 秒，商品清楚定格，人物给出一句真实购买理由。节奏快，剧情清楚，商品不能变形，不要无关剧情。
```

剧情广告必须从 `commerce_ad_subtypes.md` 中选一个有来源编号的正式小类。当前正式小类是情感反转剧情（LD-01）。不要只写“剧情广告”，也不要使用待反推名称。

## 品牌大片 Prompt 模板

```text
生成一条 15 秒竖屏品牌质感广告。商品是{商品名}，以商品图为唯一外观依据，必须保持{商品固定细节}。0-3 秒，镜头从{材质/包装/局部细节}极近特写开始，光线有质感。3-8 秒，镜头顺滑推进或环绕，展示商品完整形态。8-12 秒，商品进入{场景}，突出{品牌气质/使用氛围}。12-15 秒，商品正面干净定格。不要复杂剧情，不要人物抢戏，不要生成不可读文字。
```

品牌大片必须选择产品英雄广告（LV-05）、电影感 TV 广告（LV-06）或时尚生活方式跟拍（LV-08）中的一个。

## 达人带货 Prompt 模板

```text
生成一条 15 秒竖屏达人带货视频。商品是{商品名}，以商品图为唯一外观依据，必须保持{商品固定细节}。0-3 秒，一位自然上镜的{达人身份}在{真实场景}手持/使用商品，商品清楚出现，并抛出{痛点/结果钩子}。3-8 秒，达人完成{使用动作/试穿/开箱/演示}，证明{核心卖点}。8-12 秒，镜头给商品细节近景或使用结果。12-15 秒，达人给出一句轻行动引导。画面真实，像手机信息流，不要夸张表演。
```

达人带货必须选择 UGC 手持口播展示（LV-01）、开箱揭晓（LV-02）、使用场景证明（LV-03）、试穿前后变化（LV-04）或俯拍 ASMR 开箱（LV-07）中的一个。

## 5-10 秒短变体

```text
生成一条 10 秒竖屏{生产类型}。0-1 秒，{商品/结果/冲突/主视觉}已经出现。1-4 秒，{一个核心动作}。4-8 秒，{卖点证明或细节近景}。8-10 秒，{商品定格或轻 CTA}。只讲一个卖点，不要多场景混剪。
```
