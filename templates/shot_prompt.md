# 即梦 / Seedance 提示词模板

把选中的脚本转成可提交给 `dreamina multimodal2video` 或 `image2video` 的提示词。

## 总规则

- 默认 10-15 秒竖屏 `9:16`。
- 默认模型 `seedance2.0fast`。
- 商品图存在时优先 `multimodal2video`。
- 每条提示词必须锁定商品外观。
- 不写参考来源，不写平台名，不写“照着某视频”。
- 内部脚本必须带 `source_id`，但最终 prompt 不写来源编号。
- 不要求模型生成复杂可读文字；字幕建议后期加。
- 生成提交后如果排队，只记录任务 ID，不重复提交。
- 如果提示词来自参考视频反推，反推阶段必须同时看关键帧、字幕/画面文字、口播和镜头结构；不能只抽音频。

## 输出格式

```json
{
  "video_ratio": "9:16",
  "duration": 10,
  "production_type": "剧情广告 / 品牌大片 / 达人带货",
  "production_subtype": "",
  "source_id": "",
  "hook_structure": "信息差 / 机遇焦虑 / 视觉震撼 / 亲身经验 / 保姆级教程 / 痛点共鸣",
  "dreamina_mode": "multimodal2video / image2video / text2video",
  "model_version": "seedance2.0fast",
  "product_lock": {
    "name": "",
    "color": "",
    "material": "",
    "fixed_details": []
  },
  "prompt_file": "shots/shot_001.txt",
  "prompt": "",
  "negative_prompt": ""
}
```

## 通用 Prompt 骨架

```text
生成一条{时长}秒竖屏 9:16 的{生产类型}/{生产子类型}。商品是{商品名}，以输入商品图为唯一外观依据，必须保持{颜色、形状、材质、包装、配件、logo位置等固定细节}。

0-3 秒：{钩子画面}，商品/结果/冲突/主视觉必须已经出现。
3-8 秒：{核心动作或剧情推进}，只证明一个主卖点：{核心卖点}。
8-12 秒：{细节近景、使用结果、上身效果或反转结果}。
12-15 秒：{商品定格或轻 CTA}，商品清楚可见。

画面风格：{真实信息流 / 高级品牌片 / 短剧情广告}。
负面要求：不要品牌乱码，不要水印，不要字幕，不要改变商品外观，不要多余人物，不要复杂无关剧情。
```

## 剧情广告 Prompt 模板

```text
生成一条 15 秒竖屏剧情带货广告。商品是{商品名}，以商品图为唯一外观依据，必须保持{商品固定细节}。0-3 秒，人物在{场景}遇到{痛点/尴尬/误会}，商品或商品相关问题已经出现。3-8 秒，{商品名}自然进入剧情，人物通过{动作}发现{核心卖点}。8-12 秒，剧情出现反转：{反转结果}。12-15 秒，商品清楚定格，人物给出一句真实购买理由。节奏快，剧情清楚，商品不能变形，不要无关剧情。
```

剧情广告必须从 `commerce_ad_subtypes.md` 中选一个有来源编号的正式小类。当前正式小类包括情感反转剧情（LD-01）、场景错位短剧反转（XQ-01）、AI 工具能力展示（XQ-02）。不要只写“剧情广告”，也不要使用待反推名称。

### 场景错位短剧反转

```text
生成一条 15 秒竖屏场景错位短剧带货广告。商品是{商品名}，以商品图为唯一外观依据，必须保持{商品固定细节}。0-3 秒，在{宫廷/武侠/审讯/餐厅/办公室/家庭}这类强戏剧场景中，人物遇到{痛点/尴尬/紧急需求}，商品或需求已经清楚出现。3-6 秒，人物用{商品名}完成一个明确动作，动作要能解释{核心卖点}。6-10 秒，原本冲突被化解，人物表情从被动变成轻松、得意或掌控局面。10-13 秒，说出商品名和核心卖点。13-15 秒，用一句短促、有记忆点的收尾句强化购买理由。不要复制真实影视剧，不要让剧情盖过商品，不要把商品放到最后才出现。
```

### AI 工具能力展示

```text
生成一条 15 秒竖屏 AI 工具能力展示广告。产品是{工具名}，核心能力是{核心能力}。0-3 秒，用一个具体创作难题开场：{脚本难写/片子难拍/素材不够/复刻难做/投放素材来不及}。3-6 秒，工具出场，画面清楚展示输入一句话、上传商品图或粘贴链接。6-10 秒，展示生成过程，出现故事板、片场、手机成片、多屏结果或素材墙。10-13 秒，用画面证明{提效/批量生成/低门槛/可复刻}。13-15 秒，工具名和一句行动口号收尾。不要只拍人在讲，不要虚构真实数据，不要出现不可读界面乱码。
```

## 品牌大片 Prompt 模板

```text
生成一条 15 秒竖屏品牌质感广告。商品是{商品名}，以商品图为唯一外观依据，必须保持{商品固定细节}。0-3 秒，镜头从{材质/包装/局部细节}极近特写开始，光线有质感。3-8 秒，镜头顺滑推进或环绕，展示商品完整形态。8-12 秒，商品进入{场景}，突出{品牌气质/使用氛围}。12-15 秒，商品正面干净定格。不要复杂剧情，不要人物抢戏，不要生成不可读文字。
```

品牌大片必须选择产品英雄广告（LV-05）、电影感 TV 广告（LV-06）、时尚生活方式跟拍（LV-08）、AI 工具能力展示（XQ-02）、运动性能品牌片（XQ-03）或科技配饰冷感品牌片（XQ-04）中的一个。

### 运动性能品牌片

```text
生成一条 15 秒竖屏运动性能品牌片。商品是{商品名}，以商品图为唯一外观依据，必须保持{商品固定细节}。0-2 秒，商品参与真实运动动作，{鞋底/鞋面/装备细节/身体动作}先出现。2-6 秒，连续动作特写突出{速度/弹性/稳定/轻量/支撑}。6-11 秒，人物进入{跑道/训练场/户外路线/球场}，镜头跟随动作，商品始终可见。11-15 秒，回到商品动作或品牌短句定格。不要复杂剧情，不要虚构运动数据，不要改变商品外观。
```

### 科技配饰冷感品牌片

```text
生成一条 15 秒竖屏科技配饰冷感品牌片。商品是{商品名}，以商品图为唯一外观依据，必须保持{商品固定细节}。0-3 秒，在{空旷混凝土空间/冷色展厅/天台/未来感走廊}中出现人物和商品轮廓。3-6 秒，手拿起或佩戴{商品名}，给{材质/轮廓/镜面/金属/灯效}近景。6-10 秒，人物佩戴后进入冷感城市或极简空间，商品成为气质焦点。10-15 秒，正面特写和商品定格，强调{未来感/身份感/高级感/专业感}。不要花哨转场，不要过度口播，不要改变商品形状。
```

## 达人带货 Prompt 模板

```text
生成一条 15 秒竖屏达人带货视频。商品是{商品名}，以商品图为唯一外观依据，必须保持{商品固定细节}。0-3 秒，一位自然上镜的{达人身份}在{真实场景}手持/使用商品，商品清楚出现，并抛出{痛点/结果钩子}。3-8 秒，达人完成{使用动作/试穿/开箱/演示}，证明{核心卖点}。8-12 秒，镜头给商品细节近景或使用结果。12-15 秒，达人给出一句轻行动引导。画面真实，像手机信息流，不要夸张表演。
```

达人带货必须选择 UGC 手持口播展示（LV-01）、开箱揭晓（LV-02）、使用场景证明（LV-03）、试穿前后变化（LV-04）、俯拍 ASMR 开箱（LV-07）、食品口感连击种草（XQ-05）、礼品心意开箱（XQ-06）、套装清单展示（XQ-07）、智能小物场景连发（XQ-08）、成分背书促销（XQ-09）或睡前仪式情绪种草（XQ-10）中的一个。


## 5-10 秒短变体

```text
生成一条 10 秒竖屏{生产类型}。0-1 秒，{商品/结果/冲突/主视觉}已经出现。1-4 秒，{一个核心动作}。4-8 秒，{卖点证明或细节近景}。8-10 秒，{商品定格或轻 CTA}。只讲一个卖点，不要多场景混剪。
```
