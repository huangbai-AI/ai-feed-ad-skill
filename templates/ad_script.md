# AI 带货广告脚本模板

基于商品资料、生产大类和子类型，生成可直接转成视频提示词的脚本。

默认不依赖参考搜索。除非用户明确提供参考素材或要求搜索，否则直接用商品资料生成脚本。

生成脚本前必须读取 `templates/reference_script_library.md`。没有来源编号和参考脚本的小类不能输出。

## 输出格式

```json
{
  "recommended_type": "剧情广告 / 品牌大片 / 达人带货",
  "recommended_subtype": "",
  "source_id": "",
  "reason": "",
  "scripts": [
    {
      "type": "剧情广告 / 品牌大片 / 达人带货",
      "subtype": "",
      "source_id": "",
      "hook_structure": "信息差 / 机遇焦虑 / 视觉震撼 / 亲身经验 / 保姆级教程 / 痛点共鸣",
      "title": "",
      "target_audience": "",
      "duration": "10-15s",
      "core_selling_point": "",
      "hook": "",
      "storyboard": [
        {
          "time": "0-3s",
          "line": "",
          "visual": "",
          "purpose": "钩子 / 展示 / 证明 / 转化"
        }
      ],
      "cta": "",
      "risk_notes": []
    }
  ]
}
```

## 写作要求

- 前 1 秒必须出现商品、结果、冲突或强视觉主画面。
- 一条广告只讲 1 个主卖点。
- 每个方向都要能直接转成即梦提示词。
- 不编造价格、折扣、材质、品牌、功效。
- 有商品图时，脚本必须锁定商品外观。
- 必须读取 `templates/reference_script_library.md` 和 `templates/commerce_ad_subtypes.md`，每条脚本都要写明子类型和来源编号。
- 剧情广告要让商品自然解决冲突，不要把商品硬塞进故事。
- 品牌大片要重视光影、材质、氛围、定格，不要写成长剧情。
- 达人带货要有真实使用/手持/试穿/开箱/细节动作。
- 反推参考视频时，必须同时看关键帧、字幕/画面文字、口播和镜头结构；不能只抽音频。
- 不要输出“霸总、重生、冷暖科技风、动感潮流风、高级简约风”等待反推名称，除非先补了真实视频参考脚本。宫廷、武侠、审讯、现代服务错位场景可归入“场景错位短剧反转”，但不要作为独立小类。

## 参考话术模板

### 剧情广告

```text
开场：她以为{痛点}只能忍，直到看到{商品名}。
中段：{商品名}出场，直接改变{场景/关系/结果}。
反转：原来真正解决问题的是{核心卖点}。
结尾：{适用人群}可以试试这一款。
```

正式小类：情感反转剧情（LD-01）。

正式小类还包括：场景错位短剧反转（XQ-01）、AI 工具能力展示（XQ-02）。

场景错位短剧写法：

```text
开场：在{戏剧化场景}里，人物遇到{强冲突/紧急需求}。
动作：{商品名}通过一个清楚动作进入剧情，直接解释{核心卖点}。
反转：原本的冲突被化解，人物从被动变主动。
收尾：商品名 + 一句有网感的记忆点文案。
```

AI 工具能力展示写法：

```text
开场：{创作难题}被具体化，不要空泛说“效率低”。
输入：展示一句话、商品图或链接进入工具。
过程：出现故事板、片场、手机成片、多屏结果或素材墙。
证明：用画面证明{提效/批量生成/低门槛/可复刻}。
收尾：工具名 + 行动口号。
```

### 品牌大片

```text
开场：先给{商品细节}极近特写。
中段：镜头推进到完整商品，展示{材质/颜色/工艺/氛围}。
场景：放进{使用场景}，建立品牌气质。
结尾：商品干净定格，保留高级感。
```

正式小类：产品英雄广告（LV-05）、电影感 TV 广告（LV-06）、时尚生活方式跟拍（LV-08）、AI 工具能力展示（XQ-02）、运动性能品牌片（XQ-03）、科技配饰冷感品牌片（XQ-04）。

### 达人带货

```text
开场：如果你也有{痛点}，先看这个。
中段：我会重点看{核心卖点}，这里直接展示给你。
证明：通过{使用动作/上身效果/开箱细节}证明。
结尾：适合{目标人群}，喜欢就收藏/领券/直接拍。
```

正式小类：UGC 手持口播展示（LV-01）、开箱揭晓（LV-02）、使用场景证明（LV-03）、试穿前后变化（LV-04）、俯拍 ASMR 开箱（LV-07）、食品口感连击种草（XQ-05）、礼品心意开箱（XQ-06）、套装清单展示（XQ-07）、智能小物场景连发（XQ-08）、成分背书促销（XQ-09）、睡前仪式情绪种草（XQ-10）。

食品口感连击写法：

```text
开场：{食品场景/痛点}。
亮点：商品名 + 第一个可见卖点。
连击：3-5 个口感、分量或口味短句，每句对应一个画面。
场景：追剧、待客、嘴馋、办公室等真实吃法。
收尾：囤货、分享或复购理由。
```

礼品 / 套装写法：

```text
开场：明确送礼对象、清单焦虑或准备场景。
展示：包装、打开、逐件内容或质感细节。
证明：定制、寓意、套装齐全或心意价值。
收尾：适合谁、什么场景用、为什么省心。
```

功能小物 / 成分背书 / 睡前仪式写法：

```text
开场：一个高频麻烦、目标人群或情绪状态。
展示：商品名和真实使用动作。
证明：场景连发、成分规格或使用过程。
收尾：便利、合规 CTA 或温和状态变化。
```
