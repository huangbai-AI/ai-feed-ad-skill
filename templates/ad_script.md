# 原创广告脚本模板

基于商品资料和参考拆解，生成 3 个原创方向。

## 输出格式

```json
{
  "recommended_direction": "",
  "scripts": [
    {
      "direction": "痛点解决型 / 真人口播型 / 开箱测评型",
      "title": "",
      "target_audience": "",
      "duration": "15-30s",
      "hook": "",
      "body": [
        {
          "time": "0-3s",
          "line": "",
          "visual": ""
        }
      ],
      "cta": "",
      "why_it_should_work": ""
    }
  ]
}
```

## 写作要求

- 前 3 秒必须直接抓住痛点或结果。
- 一条广告只讲 1 个主卖点。
- 文案要像真实信息流，不要像品牌宣传片。
- 台词短，字幕短，适合手机快速刷到。
- 可以学习参考广告结构，但不要复制原句和品牌表达。
