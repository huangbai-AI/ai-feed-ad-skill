# 原创广告脚本模板

基于商品资料和参考拆解，生成 3 个原创方向。

## 输出格式

```json
{
  "recommended_direction": "",
  "scripts": [
    {
      "direction": "痛点解决型 / 真人口播型 / 开箱测评型",
      "title": "",
      "target_audience": "",
      "duration": "15-30s",
      "recommended_prompt_pattern": "",
      "pattern_reason": "",
      "hook": "",
      "body": [
        {
          "time": "0-3s",
          "line": "",
          "visual": ""
        }
      ],
      "cta": "",
      "why_it_should_work": ""
    }
  ]
}
```

## 写作要求

- 前 3 秒必须直接抓住痛点或结果。
- 一条广告只讲 1 个主卖点。
- 文案要像真实信息流，不要像品牌宣传片。
- 台词短，字幕短，适合手机快速刷到。
- 可以学习参考广告结构，但不要复制原句和品牌表达。
- 写脚本前先读取 `templates/video_prompt_patterns.md`，每个方向必须选择一个最适合的提示词模式。
- 功能型商品不要只写口播，要安排真实使用动作；服饰配饰不要只写卖点，要安排上身/上脚/佩戴效果；包装强的商品要安排产品近景或宏观英雄镜头。

## 可追加的方向

如果商品更适合视觉驱动，可以把默认 3 个方向替换或追加为：

- 开箱揭晓型：包装先制造悬念，再亮相商品和卖点。
- 使用证明型：真实场景里完成一次使用过程，让结果可见。
- 试穿变化型：前后变化制造钩子，适合服饰、鞋、包、配饰。
- 宏观质感型：用产品特写、材质、配料和慢动作提升质感。
