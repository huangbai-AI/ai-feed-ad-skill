# 商品资料整理模板

把用户输入整理为下面结构。缺失项不要阻塞，标注“待补充”。

```json
{
  "product_name": "",
  "category": "",
  "target_audience": "",
  "platform": "TikTok / 抖音 / Instagram / 小红书 / 其他",
  "pain_points": [],
  "selling_points": [],
  "usage_scenes": [],
  "price_or_offer": "",
  "proof_points": [],
  "assets": {
    "product_images": [],
    "logo": "",
    "existing_videos": []
  },
  "compliance_notes": [],
  "tone": "真实、直接、适合信息流"
}
```

整理后补一句判断：这个商品更适合“痛点解决型 / 真人口播型 / 开箱测评型 / 对比测评型”里的哪一种。
