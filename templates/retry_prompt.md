# 重试提示词模板

只修改失败镜头，不重做全部流程。

## 输入

- 原镜头提示词
- 质量检测问题
- 商品资料
- 参考广告拆解

## 输出

```json
{
  "shot_id": "",
  "retry_reason": "",
  "new_prompt": "",
  "negative_prompt": "",
  "what_changed": ""
}
```

## 改写规则

- 钩子弱：把结果、痛点或反差提前到第一秒。
- 产品不清楚：增加产品近景、手持展示、使用动作。
- 太像宣传片：加入真实口播、开箱、测评、生活场景。
- 节奏慢：减少空镜，增加动作和字幕。
- 画面不稳定：减少复杂人物和复杂场景，明确单一动作。
