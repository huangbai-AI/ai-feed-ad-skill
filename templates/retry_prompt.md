# 重试提示词模板

只修改失败镜头，不重做全部流程。重试只用于重大错误。

## 输入

- 原镜头提示词
- 重大错误检测问题
- 商品资料
- 已选广告类型和脚本方向

## 输出

```json
{
  "shot_id": "",
  "retry_reason": "",
  "new_prompt": "",
  "negative_prompt": "",
  "what_changed": ""
}
```

## 改写规则

- 只修检测出来的失败点。
- 保持原商品颜色、形状、包装、材质和关键部件。
- 保持原人物身份、服装、发型、场景和脚本方向。
- 商品不要消失、遮挡、变形，也不要新增无关道具。
- 人物不要断手断脚、多手多指、脸崩或身体结构异常。
- 字幕必须清晰可读，不要乱码、严重模糊、闪烁或穿模。
- 不要编造价格、销量、认证、医疗功效。
- 不因为钩子弱、CTA 弱、广告感不强而改方向。
