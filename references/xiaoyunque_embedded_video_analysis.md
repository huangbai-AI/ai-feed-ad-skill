## XQ-01：场景错位短剧反转

画面共性：

- 前 1 秒直接抛冲突：分手逼问、江湖大风、宴席缺酒、私藏零食被抓。
- 场景故意错位：现代商品放进餐厅、武侠、宫廷、审讯等戏剧化环境。
- 3-8 秒让商品成为破局动作：戴上降噪耳机、用送酒服务、试吃验证零食。
- 10-15 秒用一句口播把卖点钉死，常见格式是“商品名 + 结果 + 俏皮判断”。

可用口播结构：

```text
冲突质问/场景难题 -> 商品动作 -> 问题消失/局势反转 -> 商品名 -> 记忆点文案
```

参考脚本：

```text
0-3 秒：用强冲突开场，人物当场遇到难题，商品或需求已经出现在画面里。
3-6 秒：人物做一个清楚的商品动作，动作必须能解释卖点。
6-10 秒：冲突被化解或场面反转，人物表情变轻松、得意或掌控局面。
10-13 秒：说出商品名和核心卖点。
13-15 秒：用一句有网感的短句收尾，例如“废话一句都不听”“马上到”“下次多备一袋”。
```

适合：降噪耳机、外卖/到家服务、零食、清洁用品、小家电、任何能解决明确痛点的商品。

## XQ-02：AI 工具能力展示

画面共性：

- 先把创作难题戏剧化：古人说“画本创作难如登天”，或用未来感空间表现创作入口。
- 中段必须出现工具界面、生成过程、分镜 / 片场 / 手机结果，不只说“AI 很强”。
- 后段展示多结果、多题材、多屏幕，证明规模化和可复制。
- 结尾是工具标志、口号或二维码式 CTA。

参考脚本：

```text
0-3 秒：把创作难题具体化，例如“脚本难写、片子难拍、复刻难做、素材不够”。
3-6 秒：工具出场，输入一句话、商品图或链接，画面显示生成入口。
6-10 秒：展示生成过程，出现故事板、片场、手机成片或多屏结果。
10-13 秒：给出能力证明，如“一日可得多条作品”“想法自己长出画面”。
13-15 秒：工具名 + 行动口号收尾。
```

适合：AI 工具、SaaS、剪辑/生成工具、广告素材生产服务。

## XQ-03：运动性能品牌片

画面共性：

- 没有复杂口播，靠动作和节奏建立品牌感。
- 开场是鞋底、跑步动作、赛道地面等极近运动细节。
- 中段切到跑者、群跑、速度和汗水。
- 结尾品牌短句定格，例如“跑在时间前面”。

参考脚本：

```text
0-2 秒：产品参与真实运动动作，鞋底、鞋面或身体动作先出现。
2-6 秒：连续动作特写，突出速度、弹性、稳定或轻量。
6-11 秒：人物进入训练场景，与同伴或环境形成运动氛围。
11-15 秒：回到产品动作或品牌短句，形成性能记忆点。
```

## XQ-04：科技配饰冷感品牌片

画面共性：

- 冷色、极简、空旷空间，人物造型克制。
- 产品先是细节，再上身，再进入城市远景或未来感空间。
- 镜头重点是材质、轮廓、佩戴气质，不靠密集口播。

参考脚本：

```text
0-3 秒：空旷或极简场景中出现人物和产品轮廓。
3-6 秒：产品细节近景，手拿起或佩戴。
6-10 秒：人物佩戴后进入城市、天台、展厅等冷感空间。
10-15 秒：正面特写和产品定格，强调质感、未来感、身份感。
```

适合：眼镜、耳机、穿戴设备、数码配件、未来感服饰。

## XQ-05：食品口感连击种草

画面共性：

- 前 1-2 秒用痛点或品类定位：难剥、追剧饿、经典零食。
- 3-8 秒连续给大特写：开口、果仁、夹心、红油、拉丝、脆感。
- 口播用短句堆叠感官词：香、酥、脆、Q 弹、入味、料足、停不下来。
- 结尾落到场景和购买理由：追剧夜宵、过年招待、大罐耐吃。

参考脚本：

```text
0-2 秒：提出食品场景或痛点，让观众知道“现在为什么要吃它”。
2-5 秒：商品名出现，给第一个可见卖点。
5-10 秒：连续 3-5 个口感 / 分量 / 口味特写，每句都对应画面。
10-13 秒：把商品放回真实场景，如追剧、待客、嘴馋、办公室。
13-15 秒：给囤货、分享或复购理由。
```

## XQ-06：礼品心意开箱

画面共性：

- 开场直接点人群：“送女生的礼物”。
- 包装、打开、材质、颜色、定制细节逐层出现。
- 卖点不是硬参数，而是“收到的人会心动”“心意拉满”。
- 结尾扩展送礼对象：闺蜜、女友、妈妈。

参考脚本：

```text
0-3 秒：明确送礼对象和场景，包装已经出现。
3-7 秒：打开包装，展示第一眼惊喜和质感。
7-10 秒：展示颜色、材质、触感或细节。
10-13 秒：加入定制、寓意、套装完整度等心意卖点。
13-15 秒：列出适合赠送的人群或场景。
```

## XQ-07：套装清单展示

画面共性：

- 开场是“还在为清单发愁吗”，直接解决选择困难。
- 中段按件数快速列举，每个物件都有清楚画面。
- 后段用情绪价值兜底：不是一堆物品，而是一整套心意。

参考脚本：

```text
0-2 秒：提出“清单/配置/准备什么”的焦虑。
2-9 秒：按重要顺序快速展示套装内容，每件只讲一个词。
9-12 秒：强调齐全、省心、从里到外都配好。
12-15 秒：落到情绪价值或使用场景。
```

适合：嫁妆/陪嫁套装、露营套装、母婴待产包、旅行套装、节日礼盒、办公套装。

## XQ-08：智能小物场景连发

画面共性：

- 前 1 秒痛点非常具体：“别再到处找开关了”。
- 中段不是解释技术，而是连续列场景：回家、观影、放松、摸鱼、睡前。
- 画面必须对应动作：按键、磁吸、床头、手持、安装。
- 结尾把造型和生活温度连接起来。

参考脚本：

```text
0-2 秒：点出一个高频小麻烦。
2-5 秒：商品名 + 一键解决。
5-9 秒：连续展示 3 个场景，每个场景一句短口播。
9-12 秒：展示安装、收纳、携带或操作便利。
12-15 秒：用造型 / 情绪 / 生活方式收尾。
```

## XQ-09：成分背书促销

画面共性：

- 开场点目标人群：“给某类需求的人看这里”。
- 中段按升级、成分、工艺、产地、规格顺序建立信任。
- 结尾给价格 / 活动 / 稀缺性。

合规提醒：

- 不能编造治疗、改善疾病、医学认证、医生背书。
- 保健品、食品、个护产品只写用户提供的合法卖点。
- 如果用户没有提供价格、活动、产地、认证，不要自行补。

参考脚本：

```text
0-2 秒：点名目标人群或需求。
2-7 秒：展示包装和成分 / 工艺 / 规格，按用户提供信息说。
7-10 秒：给品质理由，例如产地、升级、配方或规格。
10-13 秒：给活动信息，必须来自用户输入。
13-15 秒：轻 CTA，避免医疗化承诺。
```

## XQ-10：睡前仪式情绪种草

画面共性：

- 开场不是卖参数，而是安抚情绪：“辛苦一天，别硬撑”。
- 产品变成睡前小仪式，场景是床边、暖光、独处。
- 中段展示使用方式和时长。
- 结尾落到心情和状态，而不是夸张功效。

参考脚本：

```text
0-3 秒：用情绪共鸣开场，人物疲惫或低落。
3-6 秒：商品出场，成为睡前 / 独处 / 放松仪式。
6-10 秒：展示佩戴、开启、使用过程和时长。
10-13 秒：表达舒缓、陪伴、状态切换等温和结果。
13-15 秒：产品定格或人物状态变轻松。
```

适合：美容仪、香薰、睡眠小物、眼罩、按摩仪、氛围灯。

## 总结出来的带货逻辑

1. 开场只做一件事：痛点、冲突、结果或产品主视觉，不能铺背景。
2. 商品必须早出现，最迟 3 秒内出现；短剧里也不能让剧情把商品挤到最后。
3. 每条只打一个主卖点，其他卖点只能服务这个主卖点。
4. 口播短句要跟画面一一对应，不能空喊“高级、好用、爆款”。
5. 食品和小物靠“细节连击”，短剧靠“冲突反转”，礼品靠“心意证明”，工具靠“输入到结果”。
6. 结尾不一定强促销，但必须留下购买理由、使用场景或品牌短句。
